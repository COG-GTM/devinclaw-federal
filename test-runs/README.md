# Test Runs

This directory contains validation evidence and testing resources for DevinClaw.

## Structure

### [`internal/`](./internal/)
Cognition's own validation runs proving DevinClaw works end-to-end. These are real test executions against real codebases, scored and assessed by our team.

### [`customer/`](./customer/)
Templates, guides, and examples for running your own DevinClaw validation. Start here if you want to evaluate DevinClaw against your codebase.
